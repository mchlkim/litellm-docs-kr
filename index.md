# LiteLLM 한국어 문서

이 사이트는 공식 LiteLLM 문서 구조와 디자인을 유지하면서 한국어로 읽기 쉽게 정리한 로컬 문서입니다.

LiteLLM은 OpenAI 호환 형식으로 여러 LLM 프로바이더를 호출하고, Python SDK 또는 자체 호스팅 AI Gateway(Proxy)로 운영할 수 있게 해 줍니다.
