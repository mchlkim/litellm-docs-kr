
# Agent Lightning

[Agent Lightning](https://github.com/microsoft/agent-lightning)은 코드 변경을 거의 하지 않고도 Reinforcement Learning, Automatic Prompt Optimization, Supervised Fine-tuning으로 AI 에이전트를 학습하고 최적화하는 Microsoft의 오픈소스 프레임워크입니다.

LangChain, OpenAI Agents SDK, AutoGen, CrewAI 등 모든 에이전트 프레임워크와 함께 사용할 수 있습니다. Agent Lightning은 내부적으로 LiteLLM Proxy를 사용해 LLM 요청을 라우팅하고 학습 알고리즘에 필요한 trace를 수집합니다.

- [문서](https://microsoft.github.io/agent-lightning/)
