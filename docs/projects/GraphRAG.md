
# Microsoft GraphRAG

GraphRAG는 LLM을 활용해 비정형 텍스트에서 의미 있는 구조화 데이터를 추출하는 데이터 파이프라인 및 변환 도구 모음입니다. 지식 그래프를 활용하는 그래프 기반 RAG(Retrieval-Augmented Generation) 방식을 사용해 비공개 데이터셋에 대한 추론을 개선합니다.

- [Github](https://github.com/microsoft/graphrag)
- [문서](https://microsoft.github.io/graphrag/)
- [Paper](https://arxiv.org/pdf/2404.16130)
