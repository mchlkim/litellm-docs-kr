# HolmesGPT

[HolmesGPT](https://github.com/robusta-dev/holmesgpt)는 incident 대응과 troubleshooting 과정을 개선하도록 설계된 AI 기반 observability 도구입니다. 24/7 on-call 어시스턴트처럼 Automatic Correlations, Investigations 등을 통해 alert를 더 빠르게 해결하도록 돕습니다.

LiteLLM은 HolmesGPT가 여러 LLM provider와 통합하거나 자체 모델을 가져와 self-hosting할 수 있도록 돕습니다.
