# pgai

[pgai](https://github.com/timescale/pgai)는 PostgreSQL로 RAG, semantic search, 기타 AI 애플리케이션을 더 쉽게 개발할 수 있게 해주는 도구 모음입니다.

pgai가 아직 익숙하지 않다면 [README](https://github.com/timescale/pgai)를 확인하세요.

이미 pgai에 익숙하다면 여기에서 LiteLLM 관련 문서를 확인할 수 있습니다.
- pgai의 [model calling](https://github.com/timescale/pgai/blob/main/docs/model_calling/litellm.md)에 LiteLLM 사용
- [litellm provider](https://github.com/timescale/pgai/blob/main/docs/vectorizer/api-reference.md#aiembedding_litellm)를 사용해 pgai vectorizer로 데이터 embedding을 자동 생성
