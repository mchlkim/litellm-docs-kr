# 환경 설정

데모 환경을 설정하는 데 필요한 키를 준비합니다.

모든 LLM provider에는 API 키(예: `OPENAI_API_KEY`)가 필요합니다. OpenAI, Cohere, AI21에서는 **대기 목록 없이** API 키를 받을 수 있습니다.

데모에 사용할 키를 준비해 보겠습니다.

**OpenAI**: https://platform.openai.com/account/api-keys  
**Cohere**: https://dashboard.cohere.com/welcome/login?redirect_uri=%2Fapi-keys (신용카드 필요 없음)  
**AI21**: https://studio.ai21.com/account/api-key (신용카드 필요 없음)
