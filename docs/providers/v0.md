# v0

## 개요

| 속성 | 세부 정보 |
|-------|-------|
| 설명 | v0는 코드 생성을 위해 최적화된 AI 모델을 제공하며, 특히 Next.js 애플리케이션, React 컴포넌트, 최신 웹 개발에 적합합니다. |
| LiteLLM의 Provider Route | `v0/` |
| Provider 문서 링크 | [v0 API Documentation ↗](https://v0.dev/docs/v0-model-api) |
| Base URL | `https://api.v0.dev/v1` |
| 지원되는 작업 | [`/chat/completions`](#sample-usage) |

<br />
<br />

https://v0.dev/docs/v0-model-api

**모든 v0 모델을 지원합니다. completion 요청을 보낼 때 `v0/`를 접두사로 설정하기만 하면 됩니다.**

## 사용 가능한 모델

| 모델 | 설명 | 컨텍스트 창 | 최대 출력 |
|-------|-------------|----------------|------------|
| `v0/v0-1.5-lg` | 고급 코드 생성과 추론을 위한 대형 모델 | 512,000 tokens | 512,000 tokens |
| `v0/v0-1.5-md` | 일상적인 코드 생성 작업을 위한 중형 모델 | 128,000 tokens | 128,000 tokens |
| `v0/v0-1.0-md` | 레거시 중형 모델 | 128,000 tokens | 128,000 tokens |

## 필수 변수

```python showLineNumbers title="Environment Variables"
os.environ["V0_API_KEY"] = ""  # your v0 API key from v0.dev
```

## 사용법 - LiteLLM Python SDK

### 비스트리밍

```python showLineNumbers title="v0 Non-streaming Completion"
import os
import litellm
from litellm import completion

os.environ["V0_API_KEY"] = ""  # your v0 API key

messages = [{"content": "Create a React button component with hover effects", "role": "user"}]

# v0 call
response = completion(
    model="v0/v0-1.5-md", 
    messages=messages
)

print(response)
```

### 스트리밍

```python showLineNumbers title="v0 Streaming Completion"
import os
import litellm
from litellm import completion

os.environ["V0_API_KEY"] = ""  # your v0 API key

messages = [{"content": "Create a React button component with hover effects", "role": "user"}]

# v0 call with streaming
response = completion(
    model="v0/v0-1.5-md", 
    messages=messages,
    stream=True
)

for chunk in response:
    print(chunk)
```

### 비전/멀티모달 지원

모든 v0 모델은 비전 입력을 지원하므로 텍스트와 함께 이미지를 보낼 수 있습니다.

```python showLineNumbers title="v0 Vision/Multimodal"
import os
import litellm
from litellm import completion

os.environ["V0_API_KEY"] = ""  # your v0 API key

messages = [{
    "role": "user",
    "content": [
        {
            "type": "text",
            "text": "Recreate this UI design in React"
        },
        {
            "type": "image_url",
            "image_url": {
                "url": "https://example.com/ui-design.png"
            }
        }
    ]
}]

response = completion(
    model="v0/v0-1.5-lg",
    messages=messages
)

print(response)
```

### 함수 호출

v0는 구조화된 출력을 위한 함수 호출을 지원합니다.

```python showLineNumbers title="v0 Function Calling"
import os
import litellm
from litellm import completion

os.environ["V0_API_KEY"] = ""  # your v0 API key

tools = [
    {
        "type": "function",
        "function": {
            "name": "create_component",
            "description": "Create a React component",
            "parameters": {
                "type": "object",
                "properties": {
                    "component_name": {
                        "type": "string",
                        "description": "The name of the component"
                    },
                    "props": {
                        "type": "array",
                        "items": {"type": "string"},
                        "description": "List of component props"
                    }
                },
                "required": ["component_name"]
            }
        }
    }
]

response = completion(
    model="v0/v0-1.5-md",
    messages=[{"role": "user", "content": "Create a Button component with onClick and disabled props"}],
    tools=tools,
    tool_choice="auto"
)

print(response)
```

## 사용법 - LiteLLM Proxy

LiteLLM Proxy 구성 파일에 다음을 추가하세요.

```yaml showLineNumbers title="config.yaml"
model_list:
  - model_name: v0-large
    litellm_params:
      model: v0/v0-1.5-lg
      api_key: os.environ/V0_API_KEY

  - model_name: v0-medium
    litellm_params:
      model: v0/v0-1.5-md
      api_key: os.environ/V0_API_KEY

  - model_name: v0-legacy
    litellm_params:
      model: v0/v0-1.0-md
      api_key: os.environ/V0_API_KEY
```

LiteLLM Proxy 서버를 시작하세요.

```bash showLineNumbers title="Start LiteLLM Proxy"

# RUNNING on http://0.0.0.0:4000
```

## 지원되는 OpenAI 파라미터

v0는 다음 OpenAI 호환 파라미터를 지원합니다.

| 파라미터 | 유형 | 설명 |
|-----------|------|-------------|
| `messages` | array | **필수**. 'role'과 'content'가 포함된 메시지 객체 배열 |
| `model` | string | **필수**. Model ID (v0-1.5-lg, v0-1.5-md, v0-1.0-md) |
| `stream` | boolean | 선택 사항. 스트리밍 응답 활성화 |
| `tools` | array | 선택 사항. 사용 가능한 도구/함수 목록 |
| `tool_choice` | string/object | 선택 사항. 도구/함수 호출 제어 |

참고: v0는 전체 OpenAI API와 비교해 지원되는 파라미터가 제한적입니다. `temperature`, `max_tokens`, `top_p` 같은 파라미터는 지원되지 않습니다.
