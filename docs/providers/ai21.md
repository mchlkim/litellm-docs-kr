# AI21 

LiteLLM은 다음 [AI21](https://www.ai21.com/studio/pricing) model을 지원합니다.
* `jamba-1.5-mini`
* `jamba-1.5-large`
* `j2-light`
* `j2-mid`
* `j2-ultra`


:::tip

**LiteLLM request를 보낼 때 `model=ai21/<any-model-on-ai21>` prefix만 설정하면 모든 AI21 model을 지원합니다.** 
**LiteLLM이 지원하는 전체 AI21 model은 [여기](https://models.litellm.ai)에서 확인하세요.**

:::

### API key {#api-keys}
```python
import os 
os.environ["AI21_API_KEY"] = "your-api-key"
```

## **LiteLLM Python SDK 사용법**
### Sample 사용법

```python
from litellm import completion 

# set env variable 
os.environ["AI21_API_KEY"] = "your-api-key"

messages = [{"role": "user", "content": "Write me a poem about the blue sky"}]

completion(model="ai21/jamba-1.5-mini", messages=messages)
```



## **LiteLLM Proxy Server 사용법**

LiteLLM Proxy Server로 AI21 model을 호출하는 방법은 다음과 같습니다.

1. config.yaml 수정

  ```yaml
  model_list:
    - model_name: my-model
      litellm_params:
        model: ai21/<your-model-name>  # add ai21/ prefix to route as ai21 provider
        api_key: api-key                 # api key to send your model
  ```


2. 프록시 시작 

  ```bash
  $ litellm --config /path/to/config.yaml
  ```

3. LiteLLM Proxy Server로 request 전송

  <Tabs>

  <TabItem value="openai" label="OpenAI Python v1.0.0+">

  ```python
  import openai
  client = openai.OpenAI(
      api_key="sk-1234",             # pass litellm proxy key, if you're using virtual keys
      base_url="http://0.0.0.0:4000" # litellm-proxy-base url
  )

  response = client.chat.completions.create(
      model="my-model",
      messages = [
          {
              "role": "user",
              "content": "what llm are you"
          }
      ],
  )

  print(response)
  ```
  </TabItem>

  <TabItem value="curl" label="curl">

  ```shell
  curl --location 'http://0.0.0.0:4000/chat/completions' \
      --header 'Authorization: Bearer sk-1234' \
      --header 'Content-Type: application/json' \
      --data '{
      "model": "my-model",
      "messages": [
          {
          "role": "user",
          "content": "what llm are you"
          }
      ],
  }'
  ```
  </TabItem>

## AI21 전용 parameter 전달 - `documents` {#passing-ai21-specific-parameters-documents}

LiteLLM은 모든 AI21 전용 parameter를 `litellm.completion` function에 전달할 수 있습니다. 아래는 `documents` parameter를 `litellm.completion` function에 전달하는 예시입니다.

<Tabs>

<TabItem value="python" label="LiteLLM Python SDK">

```python
response = await litellm.acompletion(
    model="jamba-1.5-large",
    messages=[{"role": "user", "content": "what does the document say"}],
    documents = [
        {
            "content": "hello world",
            "metadata": {
                "source": "google",
                "author": "ishaan"
            }
        }
    ]
)

```
</TabItem>

<TabItem value="proxy" label="LiteLLM Proxy Server">

```python
import openai
client = openai.OpenAI(
    api_key="sk-1234",             # pass litellm proxy key, if you're using virtual keys
    base_url="http://0.0.0.0:4000" # litellm-proxy-base url
)

response = client.chat.completions.create(
    model="my-model",
    messages = [
        {
            "role": "user",
            "content": "what llm are you"
        }
    ],
    extra_body = {
        "documents": [
            {
                "content": "hello world",
                "metadata": {
                    "source": "google",
                    "author": "ishaan"
                }
            }
        ]
    }
)

print(response)

```
